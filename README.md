# Useful Haskell
Useful Haskell and algorithms.

## Requirements:
* GHC: **7.10+**
* stack: **The Haskell Tool Stack**
 
## How to:
* First setup: `make setup`
* Build: `make nuild`
* Build + Run: `make run`
